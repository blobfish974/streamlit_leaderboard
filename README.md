## 🏗 Installation)

### Linux

```sh
❯ pip3 install pipenv
❯ pipenv shell
❯ pip3 install streamlit
```

And to run:
```sh
❯ pipenv shell
❯ streamlit run leaderboard.py
```
If you need to install a package:
```sh
❯ pipenv install <package_name>
```
And don't forget to add it to [requirements.txt](requirements.txt)!
